# Service to process Master/Details resources

### Consists of two repositories:
* [masterdetails-service](https://github.com/meugen/masterdetails-service) - main service implementation 
* masterdetails-compose - docker compose configuration to service up and running **You are here**
* [masterdetails-terraform-aws](https://github.com/meugen/masterdetails-terraform-aws) - terraform implementation to apply infrastructure at AWS

### Uses technologies:
* Spring Boot 3.5.7
* Spring Validations
* Spring Data JPA
* Hibernate ORM
* PostgreSQL
* Flyway
* Redis
* MapStruct
* Docker
* Docker Compose
* Terraform
* AWS / AWS SDK

### Can be tested at [masterdetails](https://masterdetails.ka.meugen.in.ua/)
